# Titanic ML Classifier

## Overview
Machine learning model to predict Titanic survival using Python and XGBoost.

## Models Used
- Logistic Regression
- Random Forest
- XGBoost (best performer - 82.21% accuracy)

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib, Seaborn

## Key Learnings
- Data cleaning and EDA
- Feature engineering
- Model comparison
- Hyperparameter tuning with GridSearchCV
- Cross validation
